# Prerequisites

    1. Docker
    2. Node.js & npm
    3. Linux Server

# Installation

1. Clone project from github:

   git clone https://github.com/awolf-lib/g_nom.git

2. Enter the project directory:
   cd .../g_nom/

3. Change config.txt to match your environment.

4. Start setup:
   ./setup.sh

# Start

Start services:
./start.sh

# Stop

Stop services:
./shutdown.sh
